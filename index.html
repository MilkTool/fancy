    Welcome to Fancy.

    Fancy is a new dynamic, pure object-oriented programming language for the Rubinius VM. It is heavily inspired by Smalltalk, Ruby, Io and Erlang. Fancy has built-in support for concurrency via the actor model, similar to Io or Erlang and allows you to use almost any Ruby library out there, including most C extensions.

    Installation

    This will get you going with the latest and greatest version of Fancy:

    $ git clone https://github.com/bakkdoor/fancy.git
$ cd fancy
$ rake

    Start it up!

    $ bin/fancy

    Or run the test suite:

    $ bin/fancy bin/fspec

    For your convenience make sure to add Fancy's bin directory to your $PATH environment variable:

    export PATH=$PATH:/path/to/fancy/bin

    If you prefer using an official release, you can install Fancy as a Ruby gem, just make sure you're running it via Rubinius:

    $ rbx -S gem install fancy

    Or use rvm / rbenv:

    $ rvm install rbx
$ rvm use rbx
$ gem install fancy
